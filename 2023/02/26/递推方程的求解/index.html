递推方程的求解

汉诺塔

递归的先把前（1,n-1) 放在中间的tmp上，然后把第n个放到c， 再递归的把中间的n-1个盘子放到C.

\(T(n) = 2T(n-1)+1\) \(T(1) = 1\) \[ T(n) = 2T(n-1)+1\\ T(n) = 2(2T(n-2)+1)+1 = 2^2T(n-2)+3\\ T(n) = 2(2(2T(n-3)+1)+1)+1 = 2^3T(n-3)+7\\ T(n) = 2(2(2(2T(n-4)+1)+1)+1)+1 = 2^4T(n-4)+15\\ T(n) = 2^{n-1}+2^{n-1}-1 = 2^n -1 \] 不断带入迭代就好 \(T(1) = T(n-(n-1))\) 所以最后一个递推方程是第\((n-1)\) 个方程
