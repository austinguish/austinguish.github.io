Solution:

  1. Start with the first word and create a new line.
  2. For each subsequent word, check if adding it to the current line would result in the line length exceeding the maximum limit . If it does, then end the current line and start a new one with the current word.
  3. If adding the word to the current line does not exceed the line length limit , then add it to the current line and continue.
  4. Once all the words have been processed, assign the remaining words to the last line.

The time complexity of this problem is .

the greedy algorithm always produces a valid solution.

Proof

  1. The algorithm starts with the first word and creates a new line. It then adds subsequent words to the current line as long as the line length limit is not exceeded.
  2. If the limit is exceeded, it ends the current line and starts a new one.
  3. Finally, it assigns any remaining words to the last line. Since the algorithm always checks if adding a word to the current line exceeds the line length limit, it ensures that no line exceeds the limit. Also, the algorithm assigns all words to some line, and each word belongs to exactly one line. Therefore, the algorithm always produces a valid solution.

The greedy algorithm is optimal.

Thus the optimal algorithm must place as many words as possible in the first of the paragraph. A moment’s reflection reveals that to maximize the number of words in the first lines, one must 1. maximize the number of words in the first lines, 2. add as many words as possible to line. If 1 were not true, then the line would have to hold the “extra” left-over words, and thus could only at most as many words as if 1 were true. Thus the greedy algorithm described above is optimal
